# [p]setgame

Ein simpler Command mit dem du das Botgame setzen kannst

## Benutzung

```bash
[p]setgame <opt1> <opt2> <gamename>

[p]setgame <play/watching/listening> <online/idle/invisible/dnd> <gamename>
```
